To compile and execute Constant program, follow the steps below. 

1. Copy Constant.c file to any directory
2. In terminal change directory (cd) to the directory where Constant.c file was copied.
3. compile program with: gcc Constant.c -lm -o myprogram.out 
4. execute program : ./myprogram.out
